# Pylog

Pylog은 Django를 사용하여 개발된 첫 번째 프로젝트로, 소셜 미디어 플랫폼을 만들기 위한 프로젝트입니다.

## 주요 기능

1. 메인 페이지: Pylog의 메인 페이지로 사용자가 게시물을 작성하고 댓글을 달 수 있는 화면입니다.
   
2. 게시물 목록: 작성된 게시물과 댓글이 간략하게 보이는 창입니다.

3. 게시물 상세 보기: 사용자가 작성한 게시물을 자세히 보고 댓글을 달 수 있는 기능이 구현된 화면입니다.
